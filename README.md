# Social-Media-User-Behavior-Dashboard

An interactive Power BI dashboard project designed to analyze social media user behavior, engagement patterns, mental health impact, and business insights using real-world inspired data.

## This project provides detailed visual analysis of:

User demographics
Platform usage trends
Daily screen time behavior
User engagement metrics
Mental health & sleep impact
Purchase behavior across platforms

##  Dashboard Features
## Dashboard 1: Overview & User Behavior
Total Users KPI
Average Daily Usage
Average Platforms Used
Average Session Duration
Gender Distribution Analysis
Top Countries by Users
Age Group Distribution
Platform Usage Comparison
Purpose of Social Media Usage
Daily Usage Hours Distribution
Key User Behavior Insights

## Dashboard 2: Engagement, Impact & Business Insights
Likes, Comments, DMs & Shares KPIs
Notification Behavior Analysis
Screen Time Concern Analysis
Mood While Scrolling
Sleep Impact Distribution
Followers vs Engagement Analysis
Purchase by Platform
Mental Health Score Gauge
Business & Engagement Insights

## Key Insights
Young users (20–30 age group) dominate social media usage
Instagram is the leading platform among users
Users spend around 3 hours daily on social media
Multi-platform behavior is common
Likes and DMs are the strongest engagement activities
Higher followers lead to increased engagement
Screen time concerns and mild sleep impact are noticeable
Instagram and TikTok drive higher purchase activity
